# GudiCafeteria

개요
============
 - 특정 지역 구내식당 리스트와 메뉴를 확인할 수 있는 Application


기간
============
 - 2018.09 ~ 2019.01


특징
============
 - Kotlin + MVVM + LiveData + Reterofit2
 - 식당 리스트 + 페이징 처리
 - 구글 맵을 활용하여 식당 리스트 표현
 - 식당 상세보기 Coornidator 적용 (배달의 민족 UI/UX 참고)
 - 식당 리뷰 추가/삭제/수정
 - 카카오 로그인 OAuth 2.0 적용
 - 서버 연동


미구현 내용
============
 - Firebase를 이용한 Push Notification


Skills
============
 - Kotlin
 - MVVM 
 - LiveData
 - Retrofit2
 - Glide
 - Git
 - Kakao Auth
 - Google Map
